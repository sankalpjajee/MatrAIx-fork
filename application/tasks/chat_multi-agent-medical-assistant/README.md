# Medical Assistant chatbot

Canonical MatrAIx chatbot task for
[Multi-Agent-Medical-Assistant](https://github.com/souvikmajumder26/Multi-Agent-Medical-Assistant).

Contributor-facing prompt docs live under `input/`. Runtime connection metadata
lives in `input/chatbot.yaml`.

Local endpoint host:

`environment/task-environments/application/chatbot-api-sidecar_multi-agent-medical-assistant`

Playground can start that sidecar (adapter + product) on `http://127.0.0.1:8902`.
Set `OPENAI_API_KEY` before the first real conversation.
